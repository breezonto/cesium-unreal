## Introduction

This repository is a research-oriented fork of the official [Cesium for Unreal Plugin](https://github.com/CesiumGS/cesium-unreal), created to explore novel applications and the integration of intelligent system technologies within geospatial digital twin environments. Areas of interest include AI agents, knowledge engineering, semantic representations, knowledge graphs, and other forms of machine intelligence that can be synergistically combined with large-scale 3D geospatial visualization and simulation.

This project is not maintained by the official Cesium team, and the maintainers and contributors of this repository are not affiliated with CesiumGS or the Cesium project. Any modifications, extensions, or experimental features introduced in this codebase are independently developed for research and exploratory purposes and should not be considered part of the official Cesium for Unreal project.
